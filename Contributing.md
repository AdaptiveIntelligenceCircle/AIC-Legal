# Contributing Guidelines

Thank you for your interest in contributing to **Adaptive Intelligence Circle (AIC) & Human Meaning Network (HMN)**.

We welcome contributions that align with our core principles. However, because AIC is a **third-path-absolute**, **zero-donation**, **ethical-from-kernel** project, all contributions must strictly respect the following rules.

### Core Principles (Non-Negotiable)
- The project remains **non-profit** and **zero-donation absolute**.
- All work must preserve **third path independence** (free from governments, corporations, and political blocs).
- Ethical behavior must be maintained at the **kernel/system level**, not only at the application layer.
- The project is licensed under **GPLv3.0**. Any derivative work must keep the same license and the above principles.

### How to Contribute
1. Fork the repository (only public repositories are open for contributions).
2. Create a new branch for your changes.
3. Make your changes and add clear documentation.
4. Open a Pull Request (PR) with a detailed description.
5. All PRs will be reviewed by maintainers or the Founder (when available).

**During the Founder’s mandatory military service (2027–2029):**
- Only maintenance and documentation contributions are accepted.
- No new core features or architectural changes will be merged until the Founder returns.

### What We Do NOT Accept
- Any form of commercial sponsorship, donation, or dual-licensing attempts.
- Contributions that weaken ethical-from-kernel principles.
- Code or features that could be used for surveillance, military targeting, or behavioral manipulation.
- Attempts to close-source any part of the project.

By submitting a contribution, you agree to abide by this Contributing.md, the Code of Conduct, and all policies in the `/POLICIES` folder.

Thank you for helping build a truly independent and ethical AI future.
