# Code of Conduct

We expect all contributors and community members to act with respect, honesty, and integrity.

**Key rules:**
- No harassment, discrimination, or personal attacks
- Respect the zero-donation and third path absolute principles
- Do not attempt to commercialize, dual-license, or close any part of the core
- Any violation may result in removal from the project

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) with the addition of AIC’s core principles above.
