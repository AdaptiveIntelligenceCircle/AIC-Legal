# 👥 Contributors & Reputation

This file lists all contributors to AIC and their earned badges under the Reputation Charter.  
Badges are awarded for meaningful contributions (code, review, governance, community).  
Points are tracked for transparency and milestone recognition.

---

## 🏅 Badge Legend
- 🥉 Bronze Contributor: 1–5 merged PRs or equivalent contributions  
- 🥈 Silver Contributor: 6–15 merged PRs or equivalent contributions  
- 🥇 Gold Contributor: 16+ merged PRs or equivalent contributions  
- 🛡️ Governance Mentor: Significant input in governance discussions  
- 📖 Documentation Champion: Major contributions to docs and community knowledge

## 🔍 Transparency
- All badges link to verifiable contributions (commit IDs, PR numbers).  
- Points are updated automatically via GitHub Actions when PRs are merged.  
- Founder or trusted reviewers validate governance/community badges.
