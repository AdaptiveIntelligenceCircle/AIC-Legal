# Fork and License Compliance Policy

**Version 1.0** — April 2026

## 1. Philosophy
We encourage forking and derivative works under the spirit of open source. However, all forks **must** strictly comply with our license and core principles.

## 2. Requirements for All Forks
Any person or organization that forks AIC repositories must:
- Retain the full GPLv3 + OpenMDW-1.0 license
- Keep all copyright notices and attribution to Adaptive Intelligence Circle
- Respect Third Path Absolute, zero-donation policy, and ethical-from-kernel principles
- Clearly state in their README that it is a fork of AIC

## 3. Monitoring
- We actively monitor forks using GitHub API and public tools.
- All fork data (username, timestamp, repository URL) is logged for compliance purposes.

## 4. Enforcement
If a fork violates the license or core principles, we reserve the right to:
- Issue a public notice
- Send formal Cease and Desist letter
- File DMCA takedown request (if applicable)
- Pursue legal action if damage is significant

By forking our repositories, you acknowledge and agree to this policy.
