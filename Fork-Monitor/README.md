# AIC-Fork-Screen

A system for monitoring and recording branches of the Adaptive Intelligence Circle (AIC) & Human Meaning Network (HMN).

Objectives:
- Record all declared forks
- Detect license violations and core principles
- Generate proof if necessary
- Maintain transparency and protect the Absolute Three Paths

## Principles

All forks must include and respect the following additional GPLv3 + OpenMDW-1.0 tricks:

- No Donation Policy

- Third Absolute Path
- Core Ethics

Violations will be published and may lead to remedial action.

## How to Use
- The `monitor-forks.py` script runs periodically (weekly).
- Data is saved to `fork-log.csv` and the `logs/` directory.
- Serious violations are logged to `COMPLIANCE-VIOLATIONS.md`.
